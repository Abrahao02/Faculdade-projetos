## 📂 Projetos

### 🔹 math-service
**Descrição:**  
O `math-service` é um microserviço REST construído em **Spring Boot**, cujo objetivo é disponibilizar operações matemáticas básicas via API RESTful.  

**Tecnologias utilizadas:**  
- Java 17  
- Spring Boot 3.5.5  
- Maven  

**Funcionalidades implementadas:**  
- Soma  
- Subtração  
- Multiplicação  
- Divisão  
- Potência
